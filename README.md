## What I'm building

I develop iOS apps independently in Swift and SwiftUI. Both are in active development in private repositories:

**Peaking** — A mountain companion app for discovering peaks, verifying summits from route data, and tracking progress across iconic hiking lists. Built on MapKit, OpenStreetMap (Overpass API), CloudKit and Strava integration. Offline-first with cached map tiles and cloud-synced progress.

**Training** — A workout planning and accountability app that compares planned sessions against real activity recorded in Apple Health. Uses HealthKit, SwiftData, EventKit and natural-language workout parsing via Foundation Models (iOS 26+).
